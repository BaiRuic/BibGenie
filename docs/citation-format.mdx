Citation Format controls how BibGenie displays Zotero items in chat context. It especially affects the item context style shown after you type `@` in the input box.

## What It Affects

- Item display in `@` search results.
- Item display after being added to chat context.
- The citation-style representation BibGenie receives as context.

<Callout type="info" title="It does not modify Zotero metadata">
  Citation Format only changes display. It does not edit item title, authors, year, journal, or other Zotero metadata.
</Callout>

## Choose a CSL Style

BibGenie uses CSL styles installed in Zotero. You can choose the default CSL style from **Citation Format** in Preferences.

Common styles include:

- APA
- Chicago
- MLA
- IEEE
- Nature
- Vancouver

## When to Change It

<Cards>
  <Card title="Writing workflows">
    Choose the style required by your field, journal, or manuscript.
  </Card>
  <Card title="Recognizing papers quickly">
    Author-year styles are often easier to scan in social sciences and humanities.
  </Card>
  <Card title="Engineering papers">
    Numbered styles may fit engineering, computer science, or IEEE-style workflows.
  </Card>
</Cards>

## FAQ

<Accordions>
  <Accordion title="Why can't I find a style?">
    BibGenie reads CSL styles installed in Zotero. Install or enable the style in Zotero first.
  </Accordion>
  <Accordion title="Does this affect Zotero's Word plugin citations?">
    No. This setting mainly affects BibGenie's chat context display.
  </Accordion>
</Accordions>
