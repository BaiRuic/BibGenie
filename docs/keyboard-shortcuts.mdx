BibGenie supports a keyboard shortcut for showing or switching the chat sidebar. You can customize it to match your workflow.

## Change the Shortcut

<Steps>
  <Step>
    Open BibGenie Preferences.
  </Step>
  <Step>
    Find the **Keyboard Shortcuts** section.
  </Step>
  <Step>
    Choose a modifier, such as Ctrl + Shift or Command + Shift.
  </Step>
  <Step>
    Enter the main key, such as `L`.
  </Step>
  <Step>
    Check for conflict warnings and save.
  </Step>
</Steps>

## Conflict Detection

BibGenie checks for conflicts with common Zotero commands. If a conflict is found, the settings page shows the conflicting command.

If the shortcut does not work, possible causes include:

- Conflict with a built-in Zotero shortcut.
- Conflict with an operating-system shortcut.
- Conflict with another Zotero plugin.
- Focus is currently inside an input, modal, or system control.
- Your input method or operating system intercepted the key combination.

## Troubleshooting

<Accordions>
  <Accordion title="The shortcut does nothing">
    Try a different combination, such as Ctrl / Command + Shift + L. Also click the main Zotero window before pressing the shortcut.
  </Accordion>
  <Accordion title="The shortcut triggers another feature">
    That usually means there is a conflict. Choose a less common main key in Keyboard Shortcuts.
  </Accordion>
  <Accordion title="Should I use Ctrl or Command on macOS?">
    Command + Shift is usually the better macOS convention.
  </Accordion>
</Accordions>
