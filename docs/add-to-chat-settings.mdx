BibGenie adds **Add to Chat** entry points in several places inside Zotero. You can turn these entry points on or off to keep the Zotero interface aligned with your workflow.

<Callout type="info" title="These toggles affect entry-point visibility">
  Disabling an Add to Chat entry point usually hides that menu item or shortcut. It does not necessarily mean BibGenie cannot add that type of context by other methods.
</Callout>

## Available Toggles

| Setting | What it controls |
| :--- | :--- |
| Library Items | Shows Add to Chat for selected Zotero items |
| Collections | Shows Add to Chat in the collection tree |
| Tags | Shows Add to Chat in the tag selector |
| Reader Tabs | Shows Add to Chat for open PDF, snapshot, and note tabs |
| PDF Text Selection | Adds selected PDF text to chat |
| EPUB Text Selection | Adds selected EPUB text to chat |
| Snapshot Text Selection | Adds selected snapshot text to chat |
| Reader Page Context | Adds context from the current PDF page |

## Recommended Defaults

If you are new to BibGenie, keep the default configuration. Common preferences:

<Cards>
  <Card title="High-frequency entries">
    Keep Library Items, Reader Tabs, and PDF Text Selection enabled.
  </Card>
  <Card title="Review workflows">
    Keep Collections and Tags enabled if you often work by project, theme, or literature group.
  </Card>
  <Card title="Cleaner menus">
    Disable lower-frequency entries if your Zotero context menus feel crowded.
  </Card>
</Cards>

## Tips

- If you read PDFs heavily, enable PDF Text Selection and Reader Page Context.
- If you write literature reviews, enable Collections and Tags.
- If you prefer a minimal setup, keep Library Items and the input-box `@` menu.
