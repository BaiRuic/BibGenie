BibGenie uses credits to meter official model usage. Credits are not the same as request counts: the cost of a request depends on the model, context length, output length, and tool usage.

<Callout type="info" title="Core product distinction">
  Free users can use core chat, context, attachments, images, screenshots, and large context. Pro adds paid web access and more monthly credits for regular research workflows. Max provides the highest monthly credits for heavy reading, comparison, and review workflows.
</Callout>

## How Credits Are Consumed

Credit usage is mainly affected by:

- Model choice: higher-cost models usually consume credits faster.
- Input length: PDFs, notes, snapshots, images, and chat history increase input tokens.
- Output length: long reviews, long tables, and full notes use more tokens.
- Tool usage: search, attachment reading, web search, and web extraction may involve multiple model calls.
- **Web Search and Web Extract**: each use of Web Search or Web Extract costs a flat **1 credit** (in addition to the model's own token consumption).
- Model multiplier: labels such as `1.5x` or `2x` provide an approximate usage hint.

<Callout type="tip" title="BYOK and local models">
  Custom models, Ollama, and LM Studio do not use BibGenie's official model service, so they do not consume BibGenie credits. Any cost or compute usage is handled by your provider or your local machine.
</Callout>

## How Many Credits Does Reading a Paper Use?

There is no single fixed number because papers, models, context size, and output requirements vary. As a practical reference:

| Scenario | Approximate usage |
| :--- | :--- |
| Translating or explaining a short selected passage | Very low |
| Summarizing a typical paper | About 1 credit |
| Deep-reading a long PDF with structured long-form output | About 2-5 credits |
| Comparing multiple papers, generating a review, or repeatedly searching and reading | Usually more than 5 credits |

## What Happens When Credits Run Out?

If your credit balance is insufficient, official model requests may not continue. You can:

- Wait for the next monthly refresh.
- Upgrade to Pro, Max, or Lifetime.
- Buy a Credit Booster.
- Use BYOK or a local model.

## Checking Your Balance

After signing in, BibGenie syncs your account status and credit balance. You can also check your balance from the website account area or the plugin user page.
